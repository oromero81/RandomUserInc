# RandomUserInc

Proyecto realizado por Oscar Romero

- Para filtar usuarios duplicados usamos el campo id.value del JSON
- Se obtienen más usuarios a medida que se hace scroll

# Stack

- SDK 24
- 100% Kotlin
- Coroutines
- Dagger-Hilt
- Picasso
- Retrofit
- OkHttp
- Gson
- ViewModels
- LiveData
- ViewBindings
- JUnit
- Mockk

# Arquitectura

- 3 módulos (Domain - Data - App)
- MVVM para presentación
